# 公務員公積金 / MPF 試算器（CSPF Calculator）

一個純前端、零後端的公積金試算工具。所有計算都在你的瀏覽器本地完成，
**不會上傳或儲存任何個人資料**，適合用來快速估算退休時的累算權益。

🔗 線上使用：https://<你的帳號>.github.io/cspf-calculator/

---

## ✨ 功能特色

- **三種身份切換**
  - **文職人員**：政府累進供款（首 3 年 5%、封頂 $1,500；第 4 年起 15%→25% 按年資遞增，無上限）
  - **紀律部隊**：可再選「員佐級 / 主任級」，各採用對應官方薪級表，並享政府額外 +2.5% 特別供款
  - **普通打工（私營 MPF）**：無政府供款，改為公司強制 5%（封頂 $1,500）+ 員工 5%
- **兩種薪資輸入方式**
  - 手動輸入月薪
  - 依官方薪級表逐年晉升（選入職薪點與最高薪點）
- **自願供款**：支援固定金額，或「工資百分比」隨薪資成長（普通打工模式）
- **視覺化圖表**
  - 權益增長曲線（X 軸隨預計退休年齡自動調整，標示 10 年歸屬期）
  - 每年每月供款堆疊圖（懸浮可看當年薪酬點、月薪、政府/公司與員工供款）
- **總薪級表參考**：內建官方 0–49 薪點對照，隨職系切換

---

## 🚀 使用方法

1. 在左側面板選擇 **職系類別**（文職 / 紀律部隊 / 普通打工）
2. 填入 **入職日期、入職年齡、預計退休年齡**
3. 選擇薪資模式：**手動輸入月薪** 或 **薪級表晉升**
4. 設定 **預期增長率、投資回報率、每月自願供款**
5. 右側即時顯示 **退休累算權益總額**、供款佔比與增長圖表
6. 將滑鼠移到圖表上，可查看每一年的細項數字

薪級表與供款比率參考香港特別行政區政府公務員事務局（截至 2026-04-01 生效）：

- 紀律人員（員佐級）薪級表：https://www.csb.gov.hk/tc_chi/admin/pay/47.html
- 紀律人員（主任級）薪級表：https://www.csb.gov.hk/tc_chi/admin/pay/46.html
- 公積金供款比率：https://www.csb.gov.hk/tc_chi/admin/retirement/421.html
